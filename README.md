# Transcend — Hero 3D Futurista

Projeto desenvolvido com **React**, **TypeScript** e **Vite**, com foco em uma experiência visual moderna, imersiva e impactante.  
A interface apresenta um **Hero Section futurista em estilo 3D**, com animações suaves, efeito de movimento com o mouse, glow neon e design premium.

---

##  Visão Geral

O **Transcend** é um projeto front-end criado para demonstrar habilidades em desenvolvimento de interfaces modernas, animações avançadas e construção de experiências visuais de alto impacto.

A proposta do projeto é entregar uma landing page com estética futurista, utilizando elementos como:

- tipografia marcante
- imagem central em destaque
- efeitos de brilho e profundidade
- animações com movimento suave
- interação baseada na posição do cursor

---

##  Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**

---

##  Funcionalidades

- Hero section com visual futurista
- Efeito de movimento 3D com interação do mouse
- Animações suaves de entrada
- Glow neon com destaque visual
- Layout moderno e responsivo
- Navegação minimalista no topo
- Experiência visual imersiva para portfólio ou landing page premium

---

##  Destaques do Projeto

Este projeto foi desenvolvido com foco em:

- **impacto visual**
- **design premium**
- **interatividade**
- **animações fluidas**
- **experiência do usuário**

É uma ótima peça para portfólio, especialmente para demonstrar domínio em interfaces criativas e modernas no front-end.
